# 🎧 Amran Music Player

A sleek and lightweight terminal-based music player designed for Termux and Linux, built with Bash, MPV, and FZF.

## ✨ Features
- 🎵 Play MP3 and M4A music files smoothly.
- ⭐ Favorite songs management (Add/Remove favorites).
- ⌨️ Fully keyboard-controlled with interactive shortcuts (Ctrl+A, Ctrl+B, Ctrl+F, Ctrl+H).
- 🚀 Fast and clean UI using FZF and MPV.
